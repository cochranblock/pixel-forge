## Full Description (4000 chars max)
Pixel Forge generates pixel art game assets using AI models that run entirely on your phone. No internet required. No subscriptions. No ads. Free forever.

Pick a category, choose a class, select a color palette, tap FORGE. Done.

WHAT YOU GET:
- 108 sprite classes across 10 categories: humanoids, creatures, monsters, weapons, equipment, consumables, terrain, nature, structures, and UI/effects
- 7 color palettes: Stardew Valley, Starbound, SNES, NES, Game Boy, PICO-8, Endesga-32
- Hybrid AI conditioning: 10 super-categories + 12 trait tags = thousands of possible combos
- Automatic device detection picks the best generation settings for your hardware
- Swipe review: rate sprites good/bad to train the quality filter
- Results in seconds. No waiting for cloud servers.

SPRITE CATEGORIES:
- Humanoid: characters, knights, elves, dwarves, heroes, goblins, zombies, cyborgs
- Creature: cats, dogs, birds, fish, insects, spiders, slimes, bears, wolves
- Monster: dragons, demons, ghosts, golems, dinosaurs, aliens
- Weapon: swords, axes, bows, guns, staves, lightsabers
- Equipment: armor, helmets, shields, cloaks, boots, keys
- Consumable: potions, food, crops, mushrooms, gems
- Terrain: floors, walls, water, lava, doors, stairs
- Nature: trees, bushes, flowers
- Structure: buildings, furniture, vehicles, mechs
- UI/FX: icons, frames, combat effects, ambient particles

HOW IT WORKS:
Pixel Forge uses a tiny diffusion model (4.2 MB) trained on 75,000+ curated pixel art tiles from CC0/CC-BY sources. The AI runs a denoising process directly on your phone's processor. Every sprite is unique, generated fresh each time.

The model uses hybrid conditioning: a super-category embedding tells it "this is a weapon" while binary trait tags specify "held item, magical, small." New combinations work without retraining.

Built with Candle ML in Rust. No Python. No TensorFlow. Pure native ARM64.

PRIVACY:
Zero data collection. Zero network calls. Zero tracking. Works in airplane mode. Your sprites stay on your device. No permissions required.

BUILT FOR GAME DEVELOPERS:
Export sprites as 32x32 PNG. Use them in your game engine (Godot, Unity, Bevy, whatever). All generated art is yours. No license restrictions on output.

OPEN SOURCE:
Model architecture and training code are public domain (Unlicense). Built by The Cochran Block LLC.
