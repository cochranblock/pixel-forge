# pixel-forge

pixel-forge is on-device AI pixel art diffusion — MoE cascade, LoRA, and 97K-param MicroUNet silos running entirely on-device. It trains and runs Karras EDM diffusion models that generate 32×32 pixel art sprites on CPU, Metal, CUDA, or Vulkan with no Python and no cloud APIs. Ships as an Android app alongside the Rust CLI. Public domain (Unlicense).
