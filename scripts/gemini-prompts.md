# Gemini Sprite Generation Prompts
#
# Instructions:
# 1. Open Gemini (gemini.google.com)
# 2. Paste one prompt at a time
# 3. Save each output image as: CLASS_batchN.png (e.g. sword_batch1.png)
# 4. Drop all images into pixel-forge/data/raw/gemini/
# 5. Run: pixel-forge ingest-gemini
#
# Each prompt generates a 6x5 grid = 30 sprites per image
# Total: ~140 prompts = ~4,200 sprites across all classes

---
